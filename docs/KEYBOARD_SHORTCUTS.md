# Resolution Hub Keyboard Shortcuts

Quick reference guide for keyboard shortcuts in the Resolution Hub.

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `?` | Show Help | Opens keyboard shortcuts modal |
| `/` | Focus Search | Jump to search box |
| `Esc` | Close Modal | Close any open modal or dropdown |
| `R` | Refresh | Refresh current view data |

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `G` then `D` | Go to Dashboard | Navigate to dashboard view |
| `G` then `C` | Go to Cases | Navigate to all cases |
| `G` then `A` | Go to Analytics | Navigate to analytics view |
| `G` then `S` | Go to Sessions | Navigate to sessions view |

## Case List View

| Shortcut | Action | Description |
|----------|--------|-------------|
| `J` | Next Case | Move selection down |
| `K` | Previous Case | Move selection up |
| `Enter` | Open Case | Open selected case modal |
| `X` | Toggle Select | Toggle checkbox for bulk actions |
| `Shift + X` | Select All | Select all visible cases |
| `Ctrl + Shift + X` | Deselect All | Clear all selections |

## Case Modal

| Shortcut | Action | Description |
|----------|--------|-------------|
| `1` | Set Pending | Change status to Pending |
| `2` | Set In Progress | Change status to In Progress |
| `3` | Set Completed | Change status to Completed (opens checklist) |
| `[` | Previous Case | Navigate to previous case |
| `]` | Next Case | Navigate to next case |
| `C` | Add Comment | Focus comment input |
| `E` | Copy Email | Copy customer email to clipboard |
| `O` | Open Order | Open Shopify order in new tab |
| `S` | View Session | Open session replay in new tab |
| `P` | View SOP | Open SOP guide in new tab |

## Bulk Actions (when cases selected)

| Shortcut | Action | Description |
|----------|--------|-------------|
| `B` then `S` | Bulk Status | Open bulk status update |
| `B` then `A` | Bulk Assign | Open bulk assign (admin only) |
| `B` then `C` | Bulk Comment | Open bulk comment dialog |
| `B` then `E` | Bulk Export | Export selected to CSV |

## Admin Only

| Shortcut | Action | Description |
|----------|--------|-------------|
| `G` then `U` | Go to Users | Navigate to user management |
| `G` then `L` | Go to Audit Log | Navigate to audit log |
| `G` then `Q` | Go to Queue | Navigate to assignment queue |

## Filters

| Shortcut | Action | Description |
|----------|--------|-------------|
| `F` then `P` | Filter Pending | Show only pending cases |
| `F` then `I` | Filter In Progress | Show only in-progress cases |
| `F` then `C` | Filter Completed | Show only completed cases |
| `F` then `A` | Filter All | Clear status filter |

## Tips

- **Combo shortcuts**: For shortcuts like `G` then `D`, press `G`, release, then press `D` within 1 second
- **Modifier keys**: `Ctrl`, `Shift`, `Alt` must be held while pressing the main key
- **Focus matters**: Some shortcuts only work when focus is not in a text input
- **Visual feedback**: Current selection is highlighted with a blue border

## Customization

Keyboard shortcuts cannot be customized at this time. Contact your administrator if you need different key bindings.
